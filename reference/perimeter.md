# Compute perimeter from polygons or other geometries

Compute perimeter from polygons or other geometries

## Usage

``` r
st_perimeter_lwgeom(x)

st_perimeter_2d(x)
```

## Arguments

- x:

  object of class `sf`, `sfc` or `sfg`

## Value

numerical vector with perimeter for each feature (geometry), with unit
of measure when possible
